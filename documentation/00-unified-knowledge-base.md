## 🎬 FFmpeg 動画制作ワークフロー（2025-08-17 確立）

### 基本コマンド構造
```bash
ffmpeg -loop 1 \
  -i "[サムネイル画像パス]" \
  -i "[音声ファイルパス]" \
  -c:v libx264 -c:a aac -shortest \
  "[出力動画パス]"
```

### パラメータ解説
- **-loop 1**: 静止画を音声時間分ループ再生
- **-c:v libx264**: H.264動画コーデック（YouTube標準）
- **-c:a aac**: AAC音声コーデック（高品質・汎用性）
- **-shortest**: 音声時間に合わせて動画終了

### バッチ処理スクリプト活用
```bash
#!/bin/bash
# 複数動画一括作成スクリプト例
for i in {1..22}; do
    ffmpeg -y -loop 1 \
      -i "thumbnail.png" \
      -i "${i}-track.wav" \
      -c:v libx264 -c:a aac -shortest \
      "${i}-track.mp4"
done
```

### 出力品質基準
- **解像度**: 1280x720 (HD)
- **フレームレート**: 25fps
- **動画コーデック**: H.264 (High 4:4:4 Predictive)
- **音声コーデック**: AAC LC, 48kHz, ステレオ, 128kbps
